# 🎬 Bootcamp de Férias 2026 — Preparação para Front-End Design Engineering

> "Não estude para decorar. Estude para entender."
>
> O objetivo deste Bootcamp é reduzir o impacto do início do próximo semestre e permitir que você chegue às primeiras aulas já familiarizado com as tecnologias que serão utilizadas ao longo das disciplinas.

---

# 📖 Apresentação

Se você concluiu o semestre anterior, provavelmente já teve contato com:

* HTML
* CSS
* JavaScript
* GitHub

Agora chegou o momento de dar o próximo passo.

Nos próximos 30 dias você conhecerá ferramentas utilizadas diariamente por desenvolvedores profissionais:

* Git e GitHub
* GitFlow (básico)
* JavaScript Moderno
* Node.js
* TypeScript
* Vite
* React
* React Router DOM
* TailwindCSS

* Deploy na Vercel

Ao final deste Bootcamp você terá desenvolvido um projeto completo chamado **CineExplorer**, um catálogo de filmes construído com tecnologias modernas do mercado.

---

# 🎯 Objetivo do Bootcamp

Preparar os alunos para o próximo semestre através da prática constante durante o período de férias.

O objetivo não é transformar você em especialista.

O objetivo é permitir que você:

✅ Entenda os conceitos básicos antes das aulas

✅ Desenvolva autonomia para estudar tecnologia

✅ Ganhe confiança ao trabalhar com React

✅ Conheça o fluxo profissional de desenvolvimento

✅ Construa um projeto para seu portfólio

# 📅 Frequência

Realize uma atividade por dia.

Se perder um dia:

✅ Continue do ponto onde parou.

❌ Não tente compensar estudando 6 horas seguidas.

Consistência é mais importante que intensidade.

---

# 🤖 Uso Responsável de IA

Ferramentas permitidas:

* ChatGPT
* GitHub Copilot
* Cursor

Utilize IA para:

✅ Tirar dúvidas

✅ Explicar conceitos

✅ Revisar código

✅ Sugerir melhorias

Não utilize IA para:

❌ Copiar projetos inteiros

❌ Entregar exercícios sem compreender

❌ Substituir o processo de aprendizagem

Regra simples:

> Se você não consegue explicar o código, então ainda não aprendeu aquele conteúdo.
